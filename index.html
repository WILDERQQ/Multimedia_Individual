Clasificación de Texturas

1 Descargar el Script

Descarga el archivo fuente haciendo clic en el botón superior derecho.

2 Instalar las Dependencias

Abre tu terminal (PowerShell, Command Prompt o Bash) e instala las librerías necesarias ejecutando el siguiente comando:

pip install numpy Pillow

3 Ejecutar el Programa

Para lanzar la interfaz de escritorio en Tkinter, ejecuta el comando desde la carpeta donde se descargó el script:

python Texturas.py
